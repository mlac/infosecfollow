4. UNPATCHABLE USBLITER8 BREAKS APPLE SECUREROM
   Vulnerabilities and Exploits · [zero-day, hardware, apple]
   Latest developments: Researchers at Paradigm Shift published
   usbliter8, a working exploit that achieves arbitrary code
   execution inside the SecureROM of Apple's A12 and A13 chips,
   code burned into silicon at manufacture that no software
   update can reach.
   The flaw permanently affects iPhones and iPads built on the
   A12 and A13 processors for as long as those devices stay in
   use. It is not a remote attack; it requires physical USB
   access to the device.
