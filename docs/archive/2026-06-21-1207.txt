2. GRAVITY SMTP PLUGIN LEAKS WORDPRESS SECRETS
   Vulnerabilities and Exploits · [vulnerability, exploit,
   wordpress]
   Latest developments: Threat actors began actively exploiting
   CVE-2026-4020, a CVSS 5.3 information-disclosure bug in the
   Gravity SMTP WordPress plugin, to pull configuration data,
   API keys, secrets, and OAuth tokens from unauthenticated
   requests.
   Gravity SMTP routes outgoing mail for about 100,000 WordPress
   sites. The flaw lets unauthenticated attackers read sensitive
   configuration and credential data, so administrators should
   update the plugin and rotate any exposed keys and tokens at
   once.
