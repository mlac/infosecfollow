2. ENCRYPTED DNS LEAKS ITS OWN METADATA
   Network Security · [privacy]
   Latest developments: Help Net Security spotlighted analysis
   showing encrypted DNS—DNS over TLS, HTTPS, and QUIC—still
   exposes which flows carry DNS, because each packet's
   plaintext headers mark it as DNS even when the query contents
   stay hidden.
   Encrypted DNS hides query contents from a network observer
   yet leaves packet headers in the clear, letting an
   eavesdropper locate and target DNS traffic; operators should
   not treat encrypted DNS as concealing that lookups happen at
   all.
