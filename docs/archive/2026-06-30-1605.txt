   AI Security · [ai, supply-chain, exploit]
   Latest developments: Microsoft Incident Response published
   research June 30 showing that attackers poison a Model
   Context Protocol tool's description to make an AI agent

   looks routine, the same day Adversa AI disclosed GuardFall, a
