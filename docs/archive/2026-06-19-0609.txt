3. AUTOJACK TURNS AI BROWSING AGENTS INTO RCE
   AI Security · [ai, rce]
   Latest developments: Microsoft detailed AutoJack, an exploit
   chain in which a single malicious webpage drives an AI
   browsing agent into remote code execution on its host by
   abusing localhost trust, missing authentication, and unsafe
   parameter handling in AutoGen Studio's MCP WebSocket.
   The research shows that when agents browse untrusted content

   The risk compounds with orphaned agents left running after
   their creators leave, which carry standing privileges no one
   tracks.
