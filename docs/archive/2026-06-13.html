4. Critical Flaws Hit Splunk and phpBB

[patch, rce]

Last 24h: Splunk patched CVE-2026-20253, a 9.8-rated unauthenticated remote code execution flaw, and phpBB fixed a 10-year-old authentication bypass that granted login as any administrator.

Splunk shipped fixes for CVE-2026-20253, a 9.8 CVSS flaw that lets an unauthenticated user create or truncate arbitrary files and reach remote code execution on Splunk Enterprise below versions 10.2.4 and 10.0.7. Ransomware and Cybercrime

5. Arch Linux AUR Hijack and NPM Lockdown

[supply-chain, malware]

Last 24h: Attackers rewrote build scripts in more than 400 Arch User Repository packages to drop a Rust infostealer and an eBPF rootkit, while npm said version 12 will stop running dependency scripts by default.

Attackers took over more than 400 packages in the Arch User Repository and rewrote their build scripts so any machine that compiled them installed a Rust credential stealer; with root, the malware loads an eBPF rootkit to hide itself. The AUR is Arch Linux's community collection, separate from the official repositories. In a parallel hardening move, npm said version 12 will stop executing dependency install scripts by default. Developers should rebuild affected systems and rotate every secret and token.

Sources: The Hacker News · BleepingComputer · SecurityWeek
