:: RANSOMWARE AND CYBERCRIME

4. ARCH LINUX AUR HIJACK  [supply-chain, malware]
   Last 24h: Attackers seized more than 400 packages in the Arch
   User Repository and rewrote their build scripts to drop a
   Rust infostealer and, with root, an eBPF rootkit.
   The AUR is Arch Linux's community-maintained package
   collection, so any developer who builds a poisoned package
   runs the malware on their own machine. The Rust binary
   harvests developer secrets, credentials, and access tokens;
   when it lands with root it loads an eBPF rootkit to hide
   itself. Arch users should audit recently built AUR packages
   and rotate any exposed secrets.
   - The Hacker News: https://thehackernews.com/2026/06/over-400-arch-linux-aur-packages.html
   - BleepingComputer: https://www.bleepingcomputer.com/news/security/over-400-arch-linux-packages-compromised-to-push-rootkit-infostealer/

5. GOOGLE SUES GEMINI-ABUSING SMISHING NETWORK  [ai, phishing]
   Last 24h: Google filed suit against a Chinese cybercrime
   network it accuses of weaponizing its Gemini AI agent to
   mass-produce phishing texts aimed at Americans.
