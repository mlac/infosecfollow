3. ARCH LINUX AUR SUPPLY-CHAIN COMPROMISE  [supply-chain,
   malware, breach]
   Last 24h: Attackers hijacked more than 400 Arch User
   Repository packages this week, rewriting build scripts to
   drop a Rust infostealer and an eBPF rootkit.
   The poisoned AUR packages install a Rust credential stealer
   on any machine that builds them, harvesting developer secrets
   and access tokens. With root, the malware loads an eBPF
   rootkit to hide itself. The AUR is Arch Linux's
   community-maintained collection, separate from official
   repositories, so users vet packages themselves. Rebuild
   affected systems and rotate every exposed credential.
   - The Hacker News: https://thehackernews.com/2026/06/over-400-arch-linux-aur-packages.html
