4. MORE THAN 400 ARCH LINUX AUR PACKAGES HIJACKED
   [supply-chain, malware]
   Last 24h: Attackers seized over 400 packages in the Arch User
   Repository this week and rewrote their build scripts to drop
   a credential stealer and rootkit.
   The Arch User Repository, Arch Linux's community-maintained
   package collection, saw more than 400 entries weaponized to
   install a Rust infostealer on any machine that built them.
   The malware harvests developer secrets and access tokens, and
   when it runs as root it loads an eBPF rootkit to hide itself.
   Developers who built affected packages should rotate every
   credential and API key, rebuild from clean sources, and
   inspect for kernel-level persistence. The incident
   underscores how community package channels carry trust that
   attackers can abuse at scale.

   nation-state]
   Last 24h: Sygnia revealed that the China-nexus group Velvet
   Ant hid inside a target's Linux login system for close to ten

   decide who may sign in, planting access where routine cleanup

   implants survived because defenders rarely audit the

   infrastructure as a prime persistence target. The case shows
   nation-state crews investing in dwell time measured in years.
   - The Hacker News: https://thehackernews.com/2026/06/china-linked-hackers-backdoored-linux.html
