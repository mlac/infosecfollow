2. ARCH LINUX AUR PACKAGE HIJACK  [supply-chain, malware,
   rootkit]
   Last 24h: Attackers seized more than 400 packages in the Arch
   User Repository and rewrote their build scripts to drop a
   credential stealer and an eBPF rootkit.
   The malware is a Rust binary that harvests developer secrets
   and access tokens from any machine that builds a poisoned
   package, and with root it loads an eBPF rootkit to hide
   itself. The AUR is Arch Linux's community-maintained
   collection, separate from the official repositories and
   unvetted by the project. The episode landed the same week npm
   announced that version 12 will stop running dependency
   scripts by default, a direct counter to this style of attack.
   Developers should rebuild affected machines and rotate every
   exposed credential.
   - The Hacker News: https://thehackernews.com/2026/06/over-400-arch-linux-aur-packages.html
   - BleepingComputer: https://www.bleepingcomputer.com/news/security/over-400-arch-linux-packages-compromised-to-push-rootkit-infostealer/
   - SecurityWeek: https://www.securityweek.com/npm-12-will-change-script-execution-behavior-to-prevent-supply-chain-attacks/

3. CRITICAL SPLUNK ENTERPRISE RCE  [patch, rce]
   Last 24h: Splunk patched CVE-2026-20253, a 9.8-severity flaw
