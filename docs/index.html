[policy, ai, export-controls]

Section 702 lets US intelligence agencies collect the communications of foreign targets abroad, and critics fault it for sweeping up Americans' data in the process. Nation-State Activity

5. Velvet Ant Backdoors Linux Login for a Decade

[apt, nation-state, backdoor]

Last 24h: Sygnia revealed that China-linked Velvet Ant backdoored the PAM and OpenSSH components of an isolated network's Linux login system and watched administrator activity for nearly ten years.

The China-nexus group Velvet Ant planted itself inside the Linux authentication stack, tampering with PAM and OpenSSH to control who could sign in and to survive routine cleanup. Sygnia says the access gave the group full visibility into administrative activity on an air-gapped network for close to a decade. The case shows how authentication infrastructure makes durable, low-noise ground for espionage. Defenders should audit PAM modules and SSH binaries for unauthorized modification.

Sources: BleepingComputer · The Hacker News

Ransomware and Cybercrime

6. Arch Linux AUR Supply-Chain Compromise

[supply-chain, infostealer, rootkit]

Last 24h: Attackers hijacked more than 400 Arch User Repository packages, rewriting build scripts to drop a Rust infostealer and, with root, an eBPF rootkit.

The AUR is Arch Linux's community package collection. Anyone who built one of the more than 400 tainted packages risked handing over developer secrets, credentials, and access tokens to a Rust binary. When it runs as root, the malware loads an eBPF rootkit to hide itself. Arch users should review recently built AUR packages and rotate exposed secrets.

Sources: The Hacker News · BleepingComputer

The suit seeks to dismantle the operation. The action signals tech firms turning to the courts as criminals fold generative AI into scam pipelines.
