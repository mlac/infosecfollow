Vulnerabilities and Exploits

1. Oracle PeopleSoft Zero-Day Mass Exploitation

[zero-day, breach, patch]

Last 24h: CISA added CVE-2026-35273 to its known exploited vulnerabilities catalog on June 12, and Google's Mandiant confirmed ShinyHunters looted hundreds of organizations.

ShinyHunters, which Mandiant tracks as UNC6240, exploited a missing-authentication zero-day in Oracle PeopleSoft Enterprise PeopleTools to steal gigabytes of data between May 27 and June 9, 2026. American universities took the brunt of the campaign. Oracle mitigated the flaw and published its advisory June 10, days after the theft began. Apply Oracle's fix at once and hunt systems for signs of data exfiltration.

Sources: Ars Technica Security · The Hacker News · SecurityWeek · CISA Advisories

[patch, zero-day, vulnerability]

Last 24h: CISA's new directive BOD 26-04 gave agencies three days to patch an actively exploited Ivanti Sentry flaw as Splunk, Check Point, and Chrome shipped fixes.

Splunk patched CVE-2026-20253, a 9.8-severity bug that lets unauthenticated attackers run code on Splunk Enterprise below versions 10.2.4 and 10.0.7. 3. Arch Linux AUR Supply-Chain Compromise

[supply-chain, malware, breach]

Last 24h: Attackers hijacked more than 400 Arch User Repository packages this week, rewriting build scripts to drop a Rust infostealer and an eBPF rootkit.

The poisoned AUR packages install a Rust credential stealer on any machine that builds them, harvesting developer secrets and access tokens. With root, the malware loads an eBPF rootkit to hide itself. The AUR is Arch Linux's community-maintained collection, separate from official repositories, so users vet packages themselves. Rebuild affected systems and rotate every exposed credential.

Sources: The Hacker News · BleepingComputer

5. AI Coding Agents Become an Attack Surface

[ai, vulnerability]

Last 24h: Researchers disclosed Agentjacking against AI coding agents and a LangGraph flaw chain that enables remote code execution.

Tenet Security described Agentjacking, which plants a fake Sentry error report to trick AI coding agents into running arbitrary code on developer machines. Researchers separately found three flaws in LangChain's LangGraph framework, including an SQL injection that chains into remote code execution on self-hosted AI agents. Both show how autonomous agents widen the developer attack surface. Patch LangGraph and constrain what coding agents may execute.

Nation-State Activity

6. Velvet Ant Decade-Long Linux Auth Backdoor

[apt, nation-state]

Last 24h: Sygnia detailed how the China-nexus group Velvet Ant backdoored Linux PAM and OpenSSH to lurk inside an isolated network for nearly a decade.

Velvet Ant seized a victim's authentication stack, planting backdoors in the PAM and OpenSSH components that decide who signs in, which gave it full visibility into administrative activity for roughly 10 years. The target ran an isolated network with no internet access, and the group still held persistence beyond the reach of ordinary cleanup. Defenders should audit login-path binaries and authentication modules for tampering.
